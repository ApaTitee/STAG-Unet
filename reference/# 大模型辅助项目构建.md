## 我的项目构建要求

项目构建基于两个阶段的划分。第一阶段为验证性开发，第二阶段为实际训练运行。

### 第一阶段：验证性开发

1. 基于 Windows 11，使用 VSCode 远程连接 WSL2(Ubuntu 24.04 LTS)，使用 git 进行版本控制，github 进行小组协作。
2. 模型语言使用 python，使用 conda 进行环境管理，本地电脑使用 RTX5060 显卡。

### 第二阶段：实际训练运行

1. 基于 VSCode SSH 远程连接 HKUST HPC4，运行训练任务。
2. 小组共 300 机时（每账号 100 机时，共 3 人）。
